# Home page

The Home Page is like your command center. It gives you an immediate sense of the census progress across all exercises.

## Sections

### Exercise Information
Displays the current census phase (e.g., "Listing" or "Main Census 2024") and the operational dates. Also included is the last time the data was updated.

### Interview Stats
High-level numbers that summarize the entire operation, such as:
*   Total Completed Cases
*   Total Partial Cases
*   Duplicate Cases

### Scorecards
Specific metrics chosen for quick monitoring. These might include:
*   Total Population Counted
*   Average Household Size
*   Enumerators active today
*   Days to completion

### Featured Indicators
Selected charts that are deemed most critical for the current phase, such as the Daily Enquiry Line or Population Pyramid.
